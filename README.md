# Proyecto de Introducción al Desarrollo de Aplicaciones Móviles

Hola, mi nombre es **Victor Lorenzo Hernandez Sosa** y soy estudiante del **Instituto Tecnológico de las Américas (ITLA)**.

Este proyecto pertenece a la materia de **Introducción al Desarrollo de Aplicaciones Móviles**, facilitada por el docente **Amadís Suárez Genao**.

Esta es una **Progressive Web App (PWA)** desarrollada explorando **React**, la cual cumple con los requisitos de la asignación:
- Página Inicial con datos personales.
- Sumadora de números.
- Traductor de números a letras (1-1000) en español.
- Tabla de multiplicar hasta el 13.
- Experiencia personal en video.
